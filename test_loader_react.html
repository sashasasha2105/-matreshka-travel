⚛️ LoaderThree React Component

1. Inline LoaderThree (только компонент)

Код:

<LoaderThree />

2. LoaderThreeDemo (компонент + текст)

Код:

<LoaderThreeDemo />

3. Fullscreen Modal (как в основном проекте)

Использование:

showFullscreenLoader() / hideFullscreenLoader()

4. Интеграция с Vanilla JavaScript

Пример кода:

// Vanilla JS может вызывать React компонент:
ReactDOM.render(
  React.createElement(LoaderThreeDemo),
  document.getElementById('target')
);
